‚ö° JavaScript

What is JavaScript?

JavaScript is a programming language that makes web pages interactive. It can change content, respond to user actions, and manipulate the DOM.

What is ECMAScript?

ECMAScript (ES) is the standard specification that JavaScript is based on. It defines the syntax, types, statements, keywords, and built-in objects that JavaScript should provide.

  • ES5 (2009): Added strict mode, JSON support, array methods like forEach, map, filter
  • ES6/ES2015 (2015): Major update with let/const, arrow functions, classes, modules, promises
  • ES2016+: Yearly releases with incremental improvements

1. Variables: var vs let vs const

Feature var (ES5) let (ES6) const (ES6)
Scope Function/Global Block Block
Hoisting Yes (undefined) Yes (TDZ) Yes (TDZ)
Redeclaration Allowed Not allowed Not allowed
Reassignment Allowed Allowed Not allowed 
// var - Function scoped, hoisted
var x = 1;
if (true) {
    var x = 2; // Same variable
}
console.log(x); // 2

// let - Block scoped
let y = 1;
if (true) {
    let y = 2; // Different variable
}
console.log(y); // 1

// const - Block scoped, cannot be reassigned
const z = 1;
// z = 2; // Error!
const obj = { name: 'John' };
obj.name = 'Jane'; // OK - object content can change

7. Functions

// Function Declaration (Hoisted)
function greet(name = "World") {
    return `Hello, ${name}!`;
}

// Function Expression (Not hoisted)
const multiply = function(a, b) {
    return a * b;
};

// Arrow Function (ES6) - Not hoisted
const add = (a, b) => a + b;

// Arrow function with block body
const processData = (data) => {
    const processed = data.map(x => x * 2);
    return processed.filter(x => x > 10);
};

// Higher-order functions
const applyOperation = (arr, operation) => {
    return arr.map(operation);
};

// IIFE (Immediately Invoked Function Expression)
(function() {
    console.log("This runs immediately");
})();

// Rest parameters (ES6)
function sum(...numbers) {
    return numbers.reduce((total, num) => total + num, 0);
}

// Destructuring parameters (ES6)
function createUser({name, age, email}) {
    return { name, age, email, id: Date.now() };
}

// Creating arrays

numbers.push(6);           // Add to end: [1,2,3,4,5,6] 
numbers.pop();             // Remove from end: [1,2,3,4,5]
numbers.unshift(0);        // Add to beginning: [0,1,2,3,4,5]

numbers.splice(2, 1, 99);  // Replace element: [1,2,99,4,5]

let evens = numbers.filter(x => x % 2 === 0);            // Filter elements
let sum = numbers.reduce((total, num) => total + num, 0); // Reduce to single value
let found = numbers.find(x => x > 3);                     // Find first match
let index = numbers.findIndex(x => x > 3);                // Find index of first match
let exists = numbers.includes(3);                         // Check if element exists

let [first, second, ...rest] = numbers;                   // Destructuring
let newArray = [...numbers, 6, 7, 8];                     // Spread operator
let flattened = [[1, 2], [3, 4]].flat();                 // Flatten array (ES2019)

11. Error Handling

// try-catch-finally
try {
    let result = riskyOperation();
    console.log(result);

1. Promises and Async/Await (ES6/ES2017)

// Promise (ES6)
function fetchData() {
    return new Promise((resolve, reject) => {
        setTimeout(() => {
            const success = Math.random() > 0.3;
            if (success) {
                resolve("Data fetched successfully!");
            } else {
                reject("Failed to fetch data");
            }
        }, 2000);
    });
}

// Using Promises
fetchData()
    .then(data => console.log(data))
    .catch(error => console.error(error))
    .finally(() => console.log("Request completed"));

// Async/Await (ES2017)
async function getData() {
    try {
        const data = await fetchData();
        console.log(data);
        return data;
    } catch (error) {
        console.error(error);
    }
}
